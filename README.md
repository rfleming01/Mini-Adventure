 # Mini Adventure
**Version: 1.0**
* Player controls move the character around the scene.
* Wizards can be interacted with to display informational on what to do.
* Pressing space clears any messages, except for the starting one.
* Fire shield can be found in the invisible maze.
* amulet can be found in the yeti cave.
* Yetis throw snowballs at the player, which freeze the player if hit enough times.
* bring the amulet to the wizard by the sea completes the game

[Go here to play the working game](https://gd.games/games/538d1bd3-c6da-416d-8598-cd95590dc96d)

*How To Compile*
* Download the current version of [Gdevelop](https://gdevelop.io/download)
* Download and unzip this source folder
* Open MiniAdventure.json in gdevelop
